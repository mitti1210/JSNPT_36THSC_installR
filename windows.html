Windows 11 Installation Guide

Follow these steps in order to set up your data analysis environment.

1
Check Processor Type

First, we need to know if your computer uses an AMD/Intel (Standard) or ARM processor. This determines which version of Rtools to install.

Method 1: Settings (Recommended)

  1. Open Settings > System > About.
  2. Look for "System type" under Device specifications.
What to look for:
If it says "x64-based processor", you have AMD/Intel.
If it says "ARM-based processor", you have ARM.

Method 2: PowerShell (Advanced)

Run the following command in PowerShell:

$Env:PROCESSOR_ARCHITECTURE

Output 'AMD64' means AMD/Intel. Output 'ARM64' means ARM.

2
Install R

Download the latest version of R from the Comprehensive R Archive Network (CRAN).

Download R for Windows

Note: Click the link "Download R x.x.x for Windows" at the top of the page. Run the installer and accept all default settings.
3
Install Rtools

Rtools is required to compile R packages from source.

Go to Rtools Download Page

Select the correct version:
  • For AMD/Intel (x64): Download Rtools44-x86_64.exe (or latest version matching R).
  • For ARM (ARM64): Download Rtools44-aarch64.exe.
[Screenshot of Rtools download page highlighting options]

Run the installer and follow the instructions. Defaults are fine.

4
Install RStudio

RStudio is the interface we will use to write R code.

Download RStudio Desktop

Scroll down to "2: Install RStudio" and click the download button for Windows.

5
Configure Settings

Let's set up the pipe operator shortcut (Ctrl + Shift + M) to use %>%.

  1. Open RStudio.
  2. Go to Tools > Global Options in the menu bar.
  3. Select Code in the sidebar.
  4. Ensure "Use native pipe operator |> (requires R 4.1+)" is UNCHECKED.
  5. Click Apply and OK.
[Screenshot of Global Options > Code settings]
6
Install Packages

We will use the pak package to speed up installation of other tools.

Copy and paste the following code into the Console pane in RStudio (usually bottom-left) and press Enter.

1. Install pak:
install.packages("pak")
2. Install Analysis Packages:
pak::pkg_install(c( "skimr", "comorbidity", "broom", "tidyverse", "here", "openxlsx", "tableone", "gtsummary", "broom.helpers", "pacman" ))

Wait for the installation to complete. It may take a few minutes.
