macOS Installation Guide

Follow these steps to set up R on your Mac.

1
Check Chip Type

We need to know if your Mac has an Intel chip or Apple Silicon (M1, M2, M3...).

  1. Click the Apple Logo in the top-left corner of your screen.
  2. Select About This Mac.
  3. Look for "Chip" or "Processor".
What to look for:
"Apple M1/M2/M3..." = Apple Silicon
"Intel Core..." = Intel
2
Install R

Download the installer that matches your chip type.

Go to R for macOS Download

Select the correct file:
  • For Apple Silicon (M1/M2/M3): Download the file ending in -arm64.pkg (e.g., R-4.x.x-arm64.pkg).
  • For Intel: Download the file ending in .pkg (e.g., R-4.x.x.pkg) under "for Intel Macs".

Open the downloaded .pkg file and follow the installer instructions.

3
Install Xcode

Xcode is required for some R packages to work correctly.

Open Xcode in App Store

Note: Xcode is a large download (10GB+). Ensure you have enough space and time. after installing, open it once to agree to the license terms.
4
Install RStudio

RStudio is the interface for using R.

Download RStudio Desktop

Scroll to "2: Install RStudio", download the version for macOS (Universal), and drag it to your Applications folder.

5
Configure Settings

Set up the pipe operator shortcut (Cmd + Shift + M) to use %>%.

  1. Open RStudio.
  2. Go to Tools > Global Options in the menu bar.
  3. Select Code in the sidebar.
  4. Ensure "Use native pipe operator |> (requires R 4.1+)" is UNCHECKED.
  5. Click Apply and OK.
[Screenshot of Global Options > Code settings]
6
Install Packages

We will use the pak package to speed up installation.

Copy and paste the following code into the Console pane in RStudio along with the Enter key.

1. Install pak:
install.packages("pak")
2. Install Analysis Packages:
pak::pkg_install(c( "skimr", "comorbidity", "broom", "tidyverse", "here", "openxlsx", "tableone", "gtsummary", "broom.helpers", "pacman" ))

Wait for the installation to complete.
